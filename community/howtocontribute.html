How to Contribute to FAF

FAF isn't being maintained by any company or business. It is being built upon the contributions of hundreds of individuals, all with different talents, giving a bit of their time and expertise to develop FAF more and more. Whether you are a programmer, graphic designer, content creator or multilingual, FAF has a space for you to use your skills for the betterment of the community!

Fixing bugs, improving performance

Game development

Want to help fix a game issue? Create your own Co-op mission? Then you can help with development on Github. To start, join our discord so that you can talk with the correct dev and invite you to our Zulip. It is recommended (but not necessary!) knowing some Lua for game developlment, Python for the server work or Java for the FAF client.

Map Development

Map development is a great way to contribute to FAF. Whether you are a beginner or an experienced map maker, you can help us create new maps for the community to enjoy. If you are new to map making, you can check out the map making guide on the wiki. If you are an experienced map maker, you can join our discord and talk with the other map makers.
